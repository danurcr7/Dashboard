Dashboard penjualan sederhana yang dikembangkan menggunakan HTML, CSS, dan JavaScript untuk menampilkan informasi penjualan dalam bentuk visual yang lebih mudah dipahami. Proyek ini dibuat untuk melatih kemampuan dalam mengolah dan menyajikan data melalui antarmuka yang interaktif serta responsif. Dashboard menampilkan beberapa metrik penjualan utama dan visualisasi data yang membantu pengguna melihat performa penjualan secara lebih cepat dan terstruktur.

Fitur:
- Menampilkan total penjualan
- Menampilkan ringkasan performa penjualan
- Visualisasi data dalam bentuk grafik
- Tampilan responsif untuk berbagai ukuran layar

Tech Stack:
- HTML
- CSS
- JavaScript

Tujuan Proyek:
Melatih kemampuan pembuatan dashboard dan visualisasi data berbasis web menggunakan teknologi frontend dasar.
